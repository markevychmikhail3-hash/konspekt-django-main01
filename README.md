# konspekt-django

Вебзастосунок для ведення конспектів занять на Django 6

Вебзастосунок, за допомогою якого користувач зможе вести конспекти своїх занять чи нотаток, як він буде робити під час вивчення якоїсь теми (предмета). Користувач після авторизації зможе створювати нові нотатки, читати та редагувати існуючи нотатки за темами.
